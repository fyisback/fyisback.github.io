UX A/B testing guide

1. Click on the "Choose elements" button (marked in yellow) and put "head" inside the text fiels (marked as green).

2. In this section, click the Add Variable button and select JavaScript. Add this piece of code (remain the same for all products)

var priceSpiderAB=document.createElement("script");
priceSpiderAB.type="text/javascript"
priceSpiderAB.src="//cdn.pricespider.com/1/lib/ps-widget.js"
priceSpiderAB.async = true;
document.body.appendChild(priceSpiderAB);

3. Now click the Add Variable button and select the HTML. Add this piece of code (remain the same for all products)

<meta name="ps-key" content="5979-635c39afca9fd0000e78d6ab">
<meta name="ps-country" content="MX">
<meta name="ps-language" content="mx">

4. Now we need to add a button. Place the div where you want to see the "Where to buy" button. Inside the "ps-sku" attribute put unique SKU code for the product

<div class="ps-widget" ps-sku="[SKU value liveshere]"></div>
